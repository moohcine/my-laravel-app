# 🚀 Internship Management System

A professional web application built with Laravel to manage internships, interns, departments, and supervisors داخل الشركة.

---

## 📌 Features

* 👥 Manage Interns (Add, Edit, Delete)
* 🏢 Manage Departments
* 🧑‍💼 Manage Supervisors
* 📊 Dashboard with statistics
* 🔍 Search and filter functionality
* 📜 History tracking system

---

## 🛠️ Technologies Used

* Laravel (PHP Framework)
* MySQL Database
* HTML, CSS, JavaScript
* Bootstrap / Tailwind (optional)

---

## ⚙️ Installation

Follow these steps to run the project locally:

```bash
git clone https://github.com/moohcine/my-laravel-app.git
cd my-laravel-app
composer install
cp .env.example .env
php artisan key:generate
```

---

## 🗄️ Database Setup

* Create a database in MySQL
* Update `.env` file:

```env
DB_DATABASE=your_database
DB_USERNAME=root
DB_PASSWORD=
```

* Run migrations:

```bash
php artisan migrate
```

---

## ▶️ Run the project

```bash
php artisan serve
```

Open in browser:
http://127.0.0.1:8000
